Commands Use In This Bot

  • +help +h : List of all Commands
  • +play ?p : Plays a song
  • +radio [radiostation] : Plays a radiostation
  • +search : Searches for 15 results
  • +nowplaying +p : Shows current song
  • +pause : Pause The Current Playing Song
  • +resume : Resume The Current Song
  • +shuffle ?mix : Shuffles the queue
  • +autoplay ?ap : *Enables autoplay - random similar songs
  • +skip ?s : Skips current song
  • +stop ?leave : Stops playing and leaves the channel
  • +volume
  • +queue +qu : Shows current Queue
  • +loop : Enables loop for off / song / queue
  • +lyircs +ly : Shows lyrics for this song (Premium)
  • +skipto : Jumps to a queue song
  • +uptime : Shows you the Bot's Uptime
